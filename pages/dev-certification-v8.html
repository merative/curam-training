C005 – Cúram V8.X Customization Certification

Test structure, objectives, and preparation advice for the V8.X Customization Certification.

Test Overview and Structure

This certification confirms that the successful candidate has the knowledge, skills, and abilities necessary to customize Cúram V8.X applications by using the Cúram Application Development Environment (ADE).

The test contains three sections that together provide a total of 40 questions.

SectionWeightTopics
Cúram Infrastructure 20% (8 questions) Product documentation and information sources, customization approach, impact analysis, localization, installation, support and upgrade
Server-Side Customization 60% (24 questions) Customizing non-source artifacts (e.g. messages and code tables), modeled classes, and non-modeled classes. Implementing RESTful and SOAP web services, deferred processes, EVX events, and batch processes
Client-Side Customization 20% (8 questions) Customizing pages, navigation, and property files. Options for customizing widgets

The passing score is 70% (28 correct questions out of 40).

Section 1 – Cúram Infrastructure

  • List the main sources of information for customizing the Cúram application.
  • Identify the relevant features for localizing applications.
  • Install, troubleshoot, and support Cúram.
  • Describe the recommended approach for customizing the out-of-the-box application.
  • Perform impact analysis to determine the changes required for customizations and Cúram version upgrades.

Section 2 – Server-Side Customization

  • Customize the following artifacts compliantly: message files, code tables, data manager files, server configuration files, and configurable validations.
  • Customize modeled classes compliantly.
  • Customize non-modeled classes compliantly.
  • Implement SOAP and RESTful web services for real-time integration.
  • Implement custom events, deferred processes, and batch jobs.

Section 3 – Client-Side Customization

  • Customize pages, navigation, and configuration properties.
  • List and describe the options for customizing widgets.

Advice and Tips

High-Level Steps

To earn C005 – Cúram V8.X Customization – Certification, follow these high-level steps:

  1. Master the material and exercises in CUR091 - Merative Cúram for Developers (Customization) 8.X by taking instructor-led or self-paced training.
  2. Review the test sections and identify the associated objectives and topics.
  3. Test your understanding by completing the mini-projects, Knowledge Assessment, and readiness mentoring (if available).
  4. Take the Practice Test (P005) to see if you are prepared for the Certification Test (C005).
  5. Register for the Certification Test.
  6. Take and pass the Certification Test.

Helpful Advice for Passing the Test

  • Create a schedule: The Customization course book contains more than 350 pages (the appendix is not examinable). Expect to spend about 30–40 hours preparing after completing the training courses. Do not leave too long a gap between completing the courses and preparing for the test.
  • Safety in numbers: If possible, set up a study group. Members often present topics to each other as a way of reviewing the material.
  • Know the test objectives: The objectives are closely aligned with the training course. Copy them to a spreadsheet and mark them off as you review and practice each skill in the ADE.
  • Be systematic: Review the lesson in the course book, complete the exercise, then review the relevant developer documentation for an alternative explanation.
  • Make summary notes: Summarise key UIM elements, build commands, and the purpose of each application navigation file.
  • Get hands-on practice: Actively practice the skills you'll be tested on. Use the lab for CUR091 or CUR092 – Merative Cúram Development Environment 8.X: Sandbox. Aim to spend as much time practicing as studying.
  • Challenge your understanding: Complete the Cúram V8.X ADE Customization Certification Preparation – Mini Project: C005_Preparation_Mini_Project.pdf
  • Are you ready? Take the Knowledge Assessment in Moodle (20 random questions from a pool of 45, up to 5 attempts). Ten sample test questions are here: C005_SampleTestQuestions.pdf. Take Practice Test P005 at least four days before the actual test.
  • When taking the test: Read all information in the question. Use prompts like "Which three…" to help select or reject options. Do a quick first pass and answer easier questions first.
  • How did you do? After the test you will receive a score report by section. If you do not pass the first time, don't give up — many people take two or more attempts.
